Tema do Projeto: Estudo sobre inadimplência de contribuintes do IPTU do Rio de Janeiro 

Objetivos do Projeto: 
•	Identificar as diversas categorias de contribuintes qualificados como inadimplentes e levantar seus valores devidos a PCRJ
	Categorias:
•	Geografia (AP, Bairro, Logradouro)
•	Tipo de imóvel (Residencial, Comercial, )
•	Valor do imóvel
•	Quantidade de contribuintes
•	Tipo de contribuinte
•	Tipo de inadimplencia

Impacto para o Órgão: 
•	Facilitar a identificação de inadimplentes
•	Identificar contas com alto valor para prefeitura
